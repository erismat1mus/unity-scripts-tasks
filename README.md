# Практические задания по Unity №6

В этом созданном мною репозитории лежат скрипты на C#, написанные в рамках выполнения заданий в Unity.

## Описание скриптов и их установка

### 1. Crickets.cs
Скрипт выводит в консоль слово «crickets» каждый кадр.
* **Как использовать:** Прикрепите этот скрипт к объекту **Main Camera**.
* **Результат:** В окне Console появится бесконечный поток сообщений (аналог стихотворения Арама Сарояна).

### 2. Destroy.cs
Скрипт предназначен для удаления объекта из иерархии сразу после запуска игры.
* **Как использовать:** Прикрепите данный скрипт к **любому объекту**, который вы хотите удалить.
* **Результат:** Сразу после нажатия кнопки Play объект исчезнет со сцены.

### 3. CreatePrimitives.cs
Скрипт автоматически создает группу кубов, выстраивая их в фигуру, напоминающую Стива из Minecraft.
* **Как использовать:** Прикрепите данный скрипт к объекту **Main Camera**.
* **Результат:** При запуске игры перед камерой в координатах (0,0,0) и выше появятся 6 кубов, образующих фигуру.
